# Support

## Where to ask questions

For usage, troubleshooting, and integration questions:

- Open a GitHub issue: `https://github.com/tokamak-network/Tokamak-zk-EVM/issues/new/choose`
- Join Tokamak Discord: `https://discord.gg/tokamak`

## Before opening an issue

Please include:

- OS and architecture
- Exact command used
- Relevant config/input file path
- Full error output

This helps maintainers reproduce and resolve problems faster.
