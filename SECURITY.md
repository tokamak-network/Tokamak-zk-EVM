# Security Policy

## Supported Branch

Security fixes are currently prioritized on:

- `main`

## Reporting a Vulnerability

Please do **not** open public GitHub issues for security vulnerabilities.

Use one of the following channels:

1. GitHub private vulnerability reporting (preferred): `https://github.com/tokamak-network/Tokamak-zk-EVM/security/advisories/new`
2. Email a maintainer: `ale@tokamak.network`

When reporting, include:

- Affected component/file path
- Reproduction steps or proof of concept
- Impact assessment (confidentiality/integrity/availability)
- Suggested mitigation (if available)

We will acknowledge receipt and triage as quickly as possible.
